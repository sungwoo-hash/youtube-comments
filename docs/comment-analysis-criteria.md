# YouTube 댓글 분석 기준 문서

> 본 문서는 YouTube 댓글 수집기의 AI 댓글 분석 기능에 적용되는 분류 기준을 설명합니다.
> 분석 엔진: Groq AI (LLaMA 3.3 70B 모델)

---

## 1. 감성 분류 기준 (긍정 / 부정 / 중립)

전체 댓글을 아래 기준에 따라 긍정 / 부정 / 중립으로 분류하고, 각 비율(%)을 산출합니다.

| 감성 | 정의 | 예시 |
|------|------|------|
| **긍정** | 영상 주제 또는 콘텐츠에 대해 공감, 동의, 지지, 감사, 칭찬을 표현하는 댓글 | "너무 유익한 영상이에요", "공감합니다", "덕분에 많이 배웠어요" |
| **부정** | 영상 주제 또는 콘텐츠에 대해 비판, 반박, 불만, 실망, 거부감을 표현하는 댓글 | "이건 현실과 다른 것 같아요", "동의하기 어렵네요", "과장된 것 같습니다" |
| **중립** | 단순 질문, 정보 공유, 개인 경험 나열 등 감성 판단이 어려운 댓글 | "저도 비슷한 경험이 있어요", "혹시 이 경우는 어떻게 하나요?", "32살 직장인입니다" |

**산출 방식:** 전체 댓글 수 대비 각 감성 댓글 수의 비율(%), 합계는 100%

---

## 2. 주요 키워드 선정 기준

댓글 전체를 분석하여 핵심 주제어 최대 10개를 선정합니다.

**선정 기준:**
- 댓글 전체에서 반복적으로 등장하는 단어 또는 주제
- 영상 주제와 직접 관련된 명사 또는 개념 중심
- 조사, 접속사, 대명사 등 의미 없는 단어는 제외

**예시:** `#재테크`, `#과소비`, `#저축`, `#투자`, `#경제`

---

## 3. 주목할 댓글 선정 기준

전체 댓글 중 대표성 있는 댓글 최대 5개를 선정합니다.

**선정 기준:**
1. 영상 주제에 대한 핵심 의견이나 대표 감정을 잘 표현한 댓글
2. 구체적인 개인 경험이나 사례를 담아 공감을 유발하는 댓글
3. 긍정 / 부정 / 중립 각 감성을 균형 있게 대표하는 댓글

---

## 4. 분석 방식

| 항목 | 내용 |
|------|------|
| 분석 대상 | 수집된 활성 댓글 전체 (삭제된 댓글 제외) |
| 처리 방식 | 80개씩 나눠 분석 후 결과 통합 |
| 일관성 | 동일한 댓글 데이터는 매번 동일한 결과 출력 (temperature=0 고정) |
| 언어 | 한국어 출력 고정 |

---

## 5. 유의사항

- AI 모델 기반 분석으로, 풍자·반어·이모지가 많은 댓글은 감성이 다르게 분류될 수 있습니다.
- 분석 결과는 참고 지표이며, 최종 판단은 담당자가 직접 댓글 내용을 확인하여 결정하시기 바랍니다.
- 동일한 CSV 파일로 분석 시 결과는 항상 동일합니다.
